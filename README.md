# Castle Tycoon Defense

نسخة ويب جاهزة للعب من المتصفح والجوال.

## تحديث ماب البحر

تم تطوير ماب البحر كمسار مستقل أولًا، بدون خلط بين مصطلحات القلعة والبحر.

### الموجود الآن

- حفظ منفصل في `localStorage`:
  - `seaProgress` لتقدم البحر.
  - `castleProgress` محفوظ للقلعة لاحقًا.
- ماب بحر بأسماء بحرية فقط: السفينة، بدن السفينة، المدافع، المحركات، الترسانة، المخزن، الرادار/البوصلة، الطاقم.
- الضغط على السفينة يفتح داخل السفينة.
- داخل السفينة يحتوي ٨ غرف ببطاقات ترقية واضحة.
- خريطة داخل السفينة قابلة للسحب على الجوال.
- قتال بحري بخمس لاينات.
- سفن عدو تأتي من اليمين.
- الإطلاق الأساسي يدوي.
- قدرات مؤقتة مع cooldown وطاقة وذخيرة محدودة:
  - وابل مدافع.
  - مساعد إطلاق مؤقت.
  - درع طوارئ.
  - تسريع محركات.
- ميني قيمز بحرية للكنوز:
  - اختيار صندوق.
  - توجيه بوصلة.
  - ربط حبال.
- واجهة جوال عمودي أولًا، وساحة القتال بدون أزرار مبعثرة.

## التشغيل

افتح `index.html` مباشرة أو فعّل GitHub Pages على المستودع.

> ملاحظة تنفيذية: المستودع الحالي يحتوي ملفات تشغيل static مباشرة، لذلك تم تنفيذ التحديث داخل `index.html` كنسخة جاهزة للتشغيل بدون حاجة إلى `npm run build`.
