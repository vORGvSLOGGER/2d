# Castle Tycoon Defense

نسخة ويب جاهزة للعب من المتصفح والجوال.

## الجديد

- خريطة قلعة محسنة برسومات وأنميشن أوضح.
- خريطة قارب البحر مع أمواج وأسماك مهاجمة.
- أزرار بناء عائمة للتحديات وبدء الهجوم بدون تغطية ساحة اللعب.
- واجهة رئيسية لاختيار الخريطة.

افتح `index.html` مباشرة أو فعّل GitHub Pages على المستودع.
