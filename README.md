# 2D Defense Game

نموذج أولي للعبة دفاع ثنائية الأبعاد.

## الفكرة

اللاعب يدافع عن قاعدة في الجهة اليسرى. الوحوش تأتي من اليمين عبر عدة مسارات. عند الضغط على أي مسار، تطلق القاعدة كرة تتدحرج وتضرب الوحوش.

## الخصائص الحالية

- خمسة مسارات لعب.
- وحوش بسرعات وصحة مختلفة.
- أسلحة تتطور مع تقدم المستوى.
- صحة للقاعدة.
- نقاط ومستويات.
- مؤثرات تصادم بسيطة.
- دعم تشغيل مباشر من المتصفح.

## التشغيل المحلي

افتح ملف:

```txt
index.html
```

في المتصفح.

## الملفات

```txt
index.html
واجهة اللعبة

game.js
منطق اللعبة والرسم والتصادم والترقيات
```
