# 2D Base Defense

لعبة دفاع ثنائية الأبعاد تعمل مباشرة في المتصفح عبر GitHub Pages.

## رابط التجربة

```txt
https://vorgvslogger.github.io/2d/
```

## النسخة المطورة

تم تحويل المشروع من تجربة بسيطة إلى لعبة بواجهة رئيسية وطورين مختلفين.

## الأطوار

### 1. الطور السريع

طور قتال مباشر. الوحوش تأتي من اليمين عبر خمسة مسارات. اللاعب يضغط على المسار لإطلاق السلاح المناسب.

### 2. Tycoon Mode

طور إدارة وتطوير. اللاعب لا يدافع فقط، بل يبني اقتصاد القاعدة:

- مصنع العملات.
- ورشة الأسلحة.
- برج تلقائي.
- بحث عسكري.
- تدريع الجدار.
- تطوير القاعدة.
- إصلاح القاعدة.

## الخصائص الحالية

- واجهة رئيسية.
- اختيار طور اللعب.
- Canvas رسومي كامل بدون ملفات خارجية.
- وحوش محسنة بصريًا.
- أسلحة مرسومة داخل اللعبة.
- موجات تصاعدية.
- Boss كل عدة موجات.
- نظام عملات.
- نظام ترقيات.
- مؤثرات ضرب وانفجار وتجميد.
- أفضل موجة محفوظة في localStorage.
- دعم الجوال والكيبورد.

## التحكم

```txt
الضغط/اللمس على المسار = إطلاق
1 إلى 5 = إطلاق في المسارات
Space = إيقاف/متابعة
R = إعادة
```

## الملفات

```txt
index.html
واجهة اللعبة، القائمة الرئيسية، CSS، وتخطيط الأطوار.

game.js
منطق اللعبة، الأطوار، الوحوش، الأسلحة، الاقتصاد، والرسم.
```
