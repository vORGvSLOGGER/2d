# 2D Base Defense

لعبة دفاع ثنائية الأبعاد تعمل مباشرة في المتصفح عبر GitHub Pages.

## رابط التجربة

```txt
https://vorgvslogger.github.io/2d/
```

## الفكرة

اللاعب يدافع عن قاعدة في الجهة اليسرى. الوحوش تأتي من اليمين عبر خمسة مسارات. كل ضغطة على مسار تطلق سلاحاً في ذلك المسار. مع العملات، يطور اللاعب القاعدة، وكل تطوير للقاعدة يفتح سلاحاً جديداً.

## النسخة الحالية

- واجهة عربية RTL.
- شاشة بداية وإعادة ومحاكاة توقف مؤقت.
- خمسة مسارات لعب.
- موجات متصاعدة.
- زعيم Boss كل عدة موجات.
- أنواع وحوش مختلفة: عدّاء، ضخم، سام، مدرع، زعيم.
- نظام عملات.
- تطوير القاعدة.
- فتح أسلحة حسب مستوى القاعدة.
- ترقية قوة السلاح.
- ترقية سرعة الإطلاق.
- إصلاح القاعدة.
- مؤثرات ضرب، انفجار، واهتزاز.
- حفظ أفضل موجة عبر localStorage.
- دعم لوحة المفاتيح والأزرار والجوال.

## التحكم

```txt
اضغط بالماوس أو اللمس على المسار للإطلاق.
1 إلى 5 = إطلاق في مسار محدد.
Space = إيقاف/متابعة.
R = إعادة.
```

## الملفات

```txt
index.html
واجهة اللعبة والتصميم.

game.js
منطق اللعب، الموجات، الوحوش، الأسلحة، التطوير، والرسم.
```

## المرحلة القادمة

- تحويل الرسومات إلى Sprite Sheets حقيقية.
- إضافة أصوات وموسيقى.
- إضافة قائمة مراحل.
- إضافة متجر دائم.
- إضافة حسابات وتخزين تقدم عبر Supabase.
- إضافة لوحة متصدرين.
