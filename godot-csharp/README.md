# Sea Tycoon Defense - C# Godot Version

هذه نسخة C# احترافية كبداية نقل من Canvas إلى Godot.

## ماذا تمت إضافته

- مشروع Godot C# داخل مجلد `godot-csharp`.
- ملف `Main.cs` يحتوي حلقة اللعب الأساسية.
- رسم سفينة اللاعب داخل الكود.
- أعداء بحرية: Fish / Raft / Submarine / Boss.
- موجات، ترقيات، ذهب، ذخيرة، درع، وقدرات.
- دعم لمس الجوال ولوحة المفاتيح.

## التشغيل

1. افتح Godot 4.3 أو أحدث - نسخة .NET.
2. اختر Import.
3. افتح ملف:

```txt
godot-csharp/project.godot
```

4. شغل المشروع.

## مهم

GitHub Pages لا يشغل C# مباشرة. الموقع الحالي HTML/Canvas يبقى للمعاينة السريعة، أما هذه النسخة فهي للفتح داخل Godot ثم التصدير Web/Android لاحقًا.
